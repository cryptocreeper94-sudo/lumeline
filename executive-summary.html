Core Capabilities

  • Real-Time Odds Ingestion — Polls 47 bookmakers every 15 minutes, capturing every line movement with timestamps
  • Source Scoring & Tiering — Each bookmaker and external source is scored on accuracy (7d/30d/90d), CLV, consistency, and timing. Sources are auto-tiered: Sharp → Reliable → Neutral → Fade
  • Anomaly Detection — 4 detectors: Synchronized Moves, Reverse Steam, House Divergence, Late Flips. Each flagged with severity and confidence
  • ML Consensus Predictions — Weighted ensemble model generates consensus predictions with confidence scores. Low-confidence games defer to house line ("House Lean" protocol)
  • Integrity Scoring — Every game gets an integrity score (0–100) based on line consistency, anomaly count, and source alignment
  • Twilio SMS Alerts — High-confidence predictions and anomaly detections pushed via SMS
